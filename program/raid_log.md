# RAID Log

Use this tracker for risks, assumptions, issues, and dependencies that affect delivery.

| RAID ID | Type | Description | Impact | Probability | Severity | Owner | Mitigation / Resolution | Due Date | Status |
|---|---|---|---|---|---|---|---|---|---|
| RAID-001 | Risk / Assumption / Issue / Dependency |  | Low / Medium / High | Low / Medium / High | Low / Medium / High |  |  | YYYY-MM-DD | Open / Watching / Resolved / Escalated |

## Escalation Rule

Escalate any item that blocks sprint completion, release approval, security signoff, production stability, or stakeholder commitment.
