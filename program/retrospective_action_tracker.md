# Retrospective Action Tracker

Use this tracker to convert retrospectives and lessons into follow-up action.

| Action ID | Source | Observation | Action | Owner | Due Date | Status | Result |
|---|---|---|---|---|---|---|---|
| RETRO-001 | Sprint / Release / Incident / Review |  |  |  | YYYY-MM-DD | New / In Progress / Done / Deferred |  |

## Retrospective Prompts

- What slowed us down?
- What improved quality?
- What caused rework?
- What should be automated?
- What should be documented?
- What should change in the next sprint?
