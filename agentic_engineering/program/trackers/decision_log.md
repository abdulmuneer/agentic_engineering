# Decision Log

Use this tracker to record important product, technical, security, and delivery decisions.

| Decision ID | Date | Area | Decision | Options Considered | Rationale | Owner | Reviewers | Impact | Revisit Date |
|---|---|---|---|---|---|---|---|---|---|
| DEC-001 | YYYY-MM-DD | Product / Architecture / Security / Delivery / UX |  |  |  |  |  |  | YYYY-MM-DD |

## Decision Rules

- Record decisions that change scope, architecture, security posture, delivery timing, or operating cost.
- Link decisions to requirements, backlog items, risks, or releases.
- Include the reason, not only the final choice.
