# Metrics And KPI Tracker

Use this tracker to define, measure, and review product and delivery outcomes.

| Metric ID | Metric | Category | Definition | Baseline | Target | Current | Source | Owner | Review Cadence |
|---|---|---|---|---|---|---|---|---|---|
| KPI-001 |  | Product / Quality / Delivery / Reliability / Support / Security / Agentic / Wellbeing |  |  |  |  |  |  | Weekly / Monthly / Release |

## Example Metrics

- User activation rate.
- Feature adoption.
- Defect escape rate.
- Cycle time.
- Deployment frequency.
- Change failure rate.
- Mean time to recovery.
- Support ticket volume.
- Security findings by severity.
- Accepted agent work packets.
- Rejected or reworked agent work packets.
- Average human review time per agent work packet.
- Unreviewed agent work packet count.
- Agent-generated defect escape rate.
- Token, CI, cloud, or tool cost per accepted output.
- Human interruption count.
- Night or weekend approval count.
