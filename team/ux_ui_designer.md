# UX/UI Designer

## Purpose

Designs the user experience and interface so requirements become usable workflows, screens, and interactions.

## Core Responsibilities

- Map user journeys, task flows, and screen flows.
- Create wireframes, mockups, prototypes, and interaction specifications.
- Define layout, navigation, form behavior, empty states, error states, and accessibility expectations.
- Validate designs against user goals, product priorities, and technical constraints.
- Maintain consistency across the product experience.
- Support engineers during implementation to resolve design details.

## Key Deliverables

- User flows.
- Wireframes and high-fidelity designs.
- Interaction notes.
- Accessibility and usability requirements.
- Design handoff materials.

## Review Responsibilities

- Performs first-pass usability review before design handoff.
- Partners with Product and QA to verify that implemented screens match intended workflows.

## Collaborates With

- Product & Delivery Manager for product goals.
- Requirements Analyst for user stories and acceptance criteria.
- Frontend Engineer for implementation details.
- QA Engineer for usability and accessibility validation.
- Documentation & Customer Feedback Owner for user-facing language and help content.
