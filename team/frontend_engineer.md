# Frontend Engineer

## Purpose

Builds the user-facing application experience based on product requirements and UX/UI design.

## Core Responsibilities

- Implement screens, components, client-side state, routing, and user interactions.
- Integrate frontend workflows with backend APIs.
- Ensure responsive behavior across supported devices and browsers.
- Implement accessibility requirements.
- Handle loading, error, empty, and permission states.
- Write frontend unit, component, and end-to-end test support where appropriate.

## Key Deliverables

- User interface implementation.
- Reusable frontend components.
- API integration code.
- Frontend test coverage.
- Accessibility and responsive behavior fixes.

## Review Responsibilities

- Performs peer review for frontend-related changes when assigned.
- Responds to review findings from the Code Quality Reviewer, UX/UI Designer, QA Engineer, and Security Reviewer.

## Collaborates With

- UX/UI Designer for design fidelity and usability.
- Backend Engineer for API contracts.
- Integration Engineer for cross-system workflows.
- QA Engineer for acceptance testing.
- Documentation & Customer Feedback Owner for user-facing copy and release notes.
