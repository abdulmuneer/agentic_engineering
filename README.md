# Agentic Engineering

This repository contains the `agentic_engineering` template.

The template is kept in its own folder so it can be copied into another project without bringing along repository-level files.

## Start Here

- [Template README](agentic_engineering/README.md) explains the full structure.
- [AGENTS template](agentic_engineering/AGENTS_template.md) describes the operating process.
- [Team roles](agentic_engineering/team) defines the role files.
- [Program trackers](agentic_engineering/program/trackers) contains intake, requirements, backlog, risk, and decision trackers.
- [Agentic workflow files](agentic_engineering/agentic) covers loops, permissions, work packets, cadence, and evals.

## Layout

```text
.
├── README.md
└── agentic_engineering/
    ├── AGENTS_template.md
    ├── agentic/
    ├── external_knowledge/
    ├── learnings/
    ├── output/
    ├── program/
    └── team/
```

Use `agentic_engineering/` as the folder to copy, adapt, or drop into another workspace.
